# Time Use Survey Analysis

Analysis of Chile's 2021 National Time Use Survey (ENUT 2021)

## Overview
This project analyzes time allocation patterns across different demographic groups, examining:
- Paid work
- Domestic work  
- Childcare (ages 0-13)
- Household care
- Personal activities
- Unpaid work

## Technical Skills Demonstrated
- **Data manipulation**: dplyr, tidyr (pivot operations, joins, grouping)
- **Data visualization**: ggplot2 (21 publication-quality plots)
- **Survey data processing**: haven (Stata files), weighted calculations
- **Code organization**: Modular structure with comprehensive documentation
- **Reproducibility**: Logging and automated output generation

## Languages & Tools
R, tidyverse, ggplot2, haven
